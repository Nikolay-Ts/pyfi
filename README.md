## How to Run

Compile the program and inside the pyfi folder there should be an .so
file. Afterward, make sure that you have a python vertial environment set up.

``` bash
python -m venv .venv
source .venv/bin/activate
```

then install the shared object into your venv

> > python -m pip install .

or if you want an instialltion that changes for each new compile automatically

> > pip -m pip install -e .
